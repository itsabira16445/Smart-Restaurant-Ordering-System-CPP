# Smart Restaurant Ordering and Billing System (C++)

## Project Overview

This project is a console-based Restaurant Ordering and Billing System developed in C++. It allows customers to order food, generates a bill, and stores order details in a text file. The project was created to practice the core concepts of C++ programming and basic file handling.

## Features

- Display restaurant menu
- Take customer orders
- Calculate total bill automatically
- Input validation for user entries
- Save order details in a text file
- Simple menu-driven interface
- Exit option

## Concepts Used

- Functions
- Loops
- Switch Case
- If-Else Statements
- File Handling
- Input Validation

## Technologies

- C++
- Visual Studio Code
- Git
- GitHub

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Compile the source file using any C++ compiler.

Example:

```bash
g++ main.cpp -o restaurant
```

Run the program:

```bash
./restaurant
```

(For Windows)

```bash
restaurant.exe
```
